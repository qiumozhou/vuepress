### 01、使用npm命令安装模块

```
npm install express      # 本地安装
npm install express -g   # 全局安装
```

### 02、查看安装信息

```
npm list -g         #查看全局安装的模块
npm list grunt      #查看某个模块的信息
```

### 03、卸载模块

```
npm update express
```

### 04、更新模块

```
npm update express
```

### 05、搜索模块

```
npm search express
```
