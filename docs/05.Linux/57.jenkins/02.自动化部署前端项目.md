### 05、修改工作目录权限
在`/home/ubuntu`下执行`chown jenkins:jenkins ubuntu`


### 06、修改nginx权限
在`/usr/sbin`下执行`chown root nginx`,`chmod u+s nginx`


### 07、添加构建命令
```
npm install 
rm -rf ./dist/*
npm run build
nginx -s reload
```
