### 01、vim安装
执行`apt-get update`,`apt-get install vim`

### 02、vim常用命令
```
esc #进入待命模式
i   #进入编辑模式
v   #进入可视化模式
:   #进入命令行模式
________________________________________________
#编辑模式

wq  #保存更改
q!  #取消保存并退出
________________________________________________
#命令行模式

set number  #调出行号
数字  #跳到指定行
________________________________________________
#待命模式

/string    # 前向搜索匹配
?string		# 反向搜索匹配
n		# 跳到下一个匹配的位置
N		# 跳到上一个匹配的位置
gg    #跳到文件开头
G     #跳到文件结尾
________________________________________________
#可视化模式

dd    #删除光标所在行
dG    #删除全部
yyp   #复制光标所在行全部内容, 在光标的下一行粘贴
yyP   #复制光标所在行全部内容, 在光标的上一行粘贴
```
